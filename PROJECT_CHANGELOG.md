\## Versioning Guidelines



Utilizzare il seguente schema:



\### Added

nuovi elementi introdotti nel progetto



\### Changed

modifiche significative a processi o struttura



\### Fixed

correzioni o miglioramenti



\### Removed

elementi eliminati dal sistema
